# 3D Orman RPG (Demo)

Basit bir 3D orman RPG demo projesi. Amaç: ormanda dolaş, gems topla, düşmanlara dikkat et.

## Nasıl çalıştırılır
1. `index.html` dosyasını ve `models/`, `sounds/` klasörlerini aynı dizine koy.
2. GitHub Pages veya basit bir HTTP sunucusu ile yayınlayın.
3. Site açıldığında "Oyunu Başlat" tuşuna basın (pointer lock etkinleşir).
